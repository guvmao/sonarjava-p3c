所有枚举类型字段都应该有Javadoc风格的注释。

说明

枚举类型的每个常量都应该有清晰的注释说明其含义，这有助于代码的可读性和可维护性。

正例

public enum TestEnum {
    /**
     * agree
     */
    agree("agree"),
    /**
     * reject
     */
    reject("reject");
    private String action;
    TestEnum(String action) {
        this.action = action;
    }
    public String getAction() {
        return action;
    }
}
