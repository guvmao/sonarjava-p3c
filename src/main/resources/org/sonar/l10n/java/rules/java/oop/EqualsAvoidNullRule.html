错例：

   public void check(String input) {
        boolean result = input.equals("expectedValue");
        if (result) {
            System.out.println("Input matches the expected value.");
        }
    }

正例：

    public void check2 () {
        boolean result = "expectedValue".equals(input);
        if (result) {
            System.out.println("Input matches the expected value.");
        }
    }

参数 allowedConstantPatterns 可用于配置语义解析失败时允许放行的常量白名单，支持正则表达式，多个表达式使用逗号分隔。

示例：StringConstants\.STRING_ONE,.*Constants\.STRING_.*
